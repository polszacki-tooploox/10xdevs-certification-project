## Target platform
- **iOS 18+ (newest iOS only)**

## Languages & frameworks
- **Swift (latest stable via newest Xcode)**
- **SwiftUI** (all UI)
- **Observation** (`@Observable`) for view-model state
- **Swift Concurrency** (`async/await`, `Task`, actors, `@MainActor`)

## Data & sync
- **SwiftData** (local persistence, offline-first)
- **CloudKit (private database)** for optional sync (enabled only after sign-in)

## Authentication
- **Sign in with Apple** via **AuthenticationServices**

## System utilities
- **OSLog** for logging/debugging

## Tooling
- **Xcode (latest stable)**
- **Swift Package Manager (SPM)** for dependencies
- **Context7** for up-to-date docs during implementation: [Context7 docs](https://context7.com/docs)
